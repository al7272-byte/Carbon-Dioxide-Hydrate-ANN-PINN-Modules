# An Introduction of Crystallizations to Undergraduate Unit Operations by Physics-Informed Neural Networks

## Overview
Two artificial intelligence modules (ANN and PINN) to predict carbon dioxide gas hydrate growth designed for classroom use in an undergraduate unit operations course.

## Contents
- Artificial Neural Network (ANN)
- Physics-Informed Neural Network (PINN)
- Experimental data from https://doi.org/10.1016/j.apenergy.2024.123664

## How to Run
(RECOMMENDED): Google Colab
1. Copy paste code from modules to a Google Colab document.
2. Redefine the “file_path” and “file_path_2” variables in your own copy of the code to match your file directory of the "Data Files" folder.
